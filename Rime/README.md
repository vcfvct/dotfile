## basic custom files for Rime
* simp(明月- 简体) only

Path should be `~/Library/Rime/` in MacOS.
In the options, `Sync user data` then `Deploy`.

Log is under `$TMPDIR/` with name like `rime.squirrel.INFO/ERROR/WARNING`.
